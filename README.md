# Hi!

This repository contains my personal dotfiles, which I use on my linux machine.
It contains dotfiles for following :
- Neovim (Editor)
- Tmux (Terminal Multiplexer)
- i3wm (Window Manager)
- alacritty (Terminal Emulator)
- dunst (Notification Daemon)
- greenclip (Clipboard Manager)
- picom (Compositor)
- polybar (Bar)
- rofi (Menu)
- starship (Shell Prompt)
